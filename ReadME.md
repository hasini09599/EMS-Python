# Employee Management System Using Python & MySQL 
## Description
Employee Management System is a Python-based application developed to manage employee records efficiently. It allows users to add, update, delete, search, and view employee details with MySQL database integration.

## Features
- Add employee records
- Update employee details
- Delete employee records
- Search employee information
- View all employee records
- Input validation for email and phone number

## Technologies Used
- Python
- MySQL
- MySQL Connector
